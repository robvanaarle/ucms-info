# UCMS Info
Infopage module for Ultimo MVC

## Requirements
* PHP 5.3
* Ultimo Config MVC
* Ultimo ORM MVC
* Ultimo Form MVC
* Ultimo Phptpl MVC
* Ultimo ACL MVC
* Ultimo Translate MVC
* Ultimo Validation MVC
* UCMS Visualiser

## Features
* List, add, update and delete info pages
* Images
