# UCMS Info
Infopage module for Ultimo MVC

## Requirements
* PHP 5.3
* Ultimo MVC Config
* Ultimo MVC ORM
* Ultimo MVC Form
* Ultimo MVC Phptpl
* Ultimo MVC ACL
* Ultimo MVC Translate
* Ultimo MVC Validation
* UCMS Visualiser

## Features
* List, add, update and delete info pages
* Images
